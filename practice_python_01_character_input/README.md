Character Input - Exercise 01/40 (https://www.practicepython.org/exercise/2014/01/29/01-character-input.html)

Overview:
This simple Python program asks for the user's name and age, then calculates and outputs the year when they will turn 100 years old.

Learning take aways:
- Take user input with input()
- Convert strings to integers with int() and make code more compact by writing as int(input())
- Working with datetime module to get the current year
- Basic math operations in py
- String formatting and output

Future Improvements:
- Handle non-numeric age inputs
- Add a way for the user to check other milestones
- Add calculations so that birth month and day could be included.
